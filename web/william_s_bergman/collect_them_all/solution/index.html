Each image comes from a Wacky Packages trading card. Determine the way in which each Wacky Packages name has been changed from the original brand name. For example, “Band-Aid” has become “Bald-Aid”, meaning that N has been replaced by L; “Comet” has become “Commie”, meaning that ET has been replaced with MIE; “Deer Park” has become “Deer Parts”, meaning that K has been replaced by TS.

Reverse each replacement on the given string; thus, replace the L with N, then replace the MIE with ET, then replace the TS with K, etc. In each case there will be a single substring to replace. Doing this reveals the clue phrase NRFB STANDS FOR WHAT. NRFB is a collectibles term for NEVER REMOVED FROM BOX.
