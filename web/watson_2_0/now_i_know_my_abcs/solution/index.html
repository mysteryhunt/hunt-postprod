The title hints at the alphabet song, which provides a mapping from letters to notes: A↔Do, B↔Do, C↔Sol, D↔Sol, E↔La, F↔La, G↔Sol, H↔Fa, I↔Fa, J↔Mi, K↔Mi, etc.

The way each clue works is as follows. The word clue portion usually contains two separate clues for two words, with a “+” symbol to denote concatenation of the two answers. When the word clue(s) are solved correctly, the total number of letters in the concatenated answers exactly fits the number of notes in the associated image. The circle in each musical note is to be filled in with a letter. (Note that the musical notes requiring filled-in black circles have been shaded light gray instead, so that it’s clear you can still write a letter inside those circles.) Furthermore, the correct letters, when played/hummed using the alphabet song’s letter-to-note mapping and using the given note/rest lengths, produce a possibly-transposed version of a recognizable tune. Finally, the number in parentheses in large font at the end of each clue is an index into the name of the tune for extracting a letter.
